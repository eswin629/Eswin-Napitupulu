Scenario 1: Access a Product via category after applying multiple filters
• Go to www.ebay.com
• Navigate to Search by category > Electronics > Cell Phones & accessories
• After the page loads, click Cell Phones & Smartphones in the left hand
side navigation section.
• Now, click – All Filters (appears at the end of the filter drop downs)
• Add 3 filters - Condition, Price and Item location appearing in the pop-up and
click apply.
• Verify that the filter tags are applied.
Scenario 2: Access a Product via Search
• Go to www.ebay.com
• Type any search string in the search bar. For example: MacBook.
• Change the Search category. For example: Computers/Tablets & Networking
and click Search.
• Verify that the page loads completely.
• Verify that the first result name matches with the search string.

Pengujian menggunakan tools katalon.
Skenario 1 dan 2 berhasil dijalankan.
